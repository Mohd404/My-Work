# To create a repo from local:
```
cd /github-projects

mkdir (file name)

git init                                  

touch (file name).(type)

git add (file name)

git commit -m "Message"

Go to GitHub create a repo

git branch -M master

git remote add origin git@github.com:Mohd404/Courses.get

git push -u origin master 
```
